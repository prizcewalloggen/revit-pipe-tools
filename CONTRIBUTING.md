# Contributing to Pipe Length Calculator

Thank you for considering contributing to this Revit plugin!

## How to Contribute

### Reporting Bugs
1. Check if the bug has already been reported in Issues
2. If not, create a new issue with:
   - Clear description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Revit version
   - Screenshots if applicable

### Suggesting Features
1. Create an issue describing:
   - The feature you'd like to see
   - Why it would be useful
   - How you envision it working

### Code Contributions

#### Setup Development Environment
1. Fork the repository
2. Clone your fork
3. Install:
   - Visual Studio 2019 or later
   - Revit 2024 or later
   - Revit 2024 SDK

#### Making Changes
1. Create a new branch: `git checkout -b feature/your-feature-name`
2. Make your changes
3. Test thoroughly in Revit
4. Commit with clear messages
5. Push to your fork
6. Create a Pull Request

#### Code Style
- Use meaningful variable names
- Add comments for complex logic
- Follow C# naming conventions
- Keep methods focused and concise

#### Testing
- Test with various piping configurations
- Test with different Revit versions if possible
- Include screenshots of results

### Pull Request Process
1. Update README.md if needed
2. Describe what your PR does
3. Link any related issues
4. Wait for review

## Questions?
Feel free to open an issue for any questions about contributing!
